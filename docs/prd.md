## 4. Technical Requirements

### 4.1 HTML Standards
- Semantic HTML5 markup
- Accessible with ARIA attributes where appropriate
- Mobile-first responsive design
- Validated code with no console errors
- Meaningful H1-H3 headings with SEO keywords
- No duplicate H1 tags per page
- Comment at top of each file describing purpose

### 4.2 CSS Requirements
- Modern, simple CSS without frameworks
- Mobile-first approach
- Critical CSS included
- Compact and organized structure
- High-contrast CTA buttons
- Hover effects for service cards
- System fonts or Google Fonts with fallback
\n### 4.3 JavaScript Requirements
- Vanilla JavaScript only
- No external dependencies that break offline
- Smooth scroll to anchors
- Currency toggle functionality (GBP/USD)
- Mobile hamburger menu
- Back-to-top button
- Defer non-critical JS

### 4.4 SEO Optimization
- Meta tags: title, description, canonical, viewport, charset
- Open Graph (og:) tags for social sharing
- Twitter Card tags\n- Canonical links pointing to https://asquaredigitalmarketing.com/
- Meta robots tag:'index, follow'
- Primary keywords: 'Digital Marketing Agency UK', 'SEO Agency UK', 'PPC Agency UK', 'Performance Marketing UK', 'Lead Generation UK'
- Sample home meta description: 'A Square Digital Marketing — premium UK digital marketing agency focused on SEO, PPC, performance marketing, and lead generation. Get a free audit and grow online.'
\n### 4.5 Structured Data (JSON-LD)
- Organization schema\n- LocalBusiness schema (UK location)
- BreadcrumbList schema
- Website schema
- BlogPosting schema for blog post

### 4.6 Performance Optimization
- Lazy-load images with loading='lazy' attribute
- Responsive images with srcset placeholders
- Alt text for all images
- Optimized file sizes
\n### 4.7 Additional Files
- sitemap.xml with all pages listed
- robots.txt with appropriate directives
- README with deployment notes, customization guide, and SEO tips
